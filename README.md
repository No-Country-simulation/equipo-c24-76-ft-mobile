# Rumbo: Conéctate y Comparte Aventuras

![Rumbo Logo](assets/logo.png) 

## Descripción

**Rumbo** es una aplicación móvil desarrollada con Flutter que permite a los mochileros y viajeros compartir sus experiencias, recomendaciones y fotos en una plataforma social dinámica. Diseñada para conectar a personas que buscan aventuras y opciones de transporte, Rumbo es el lugar perfecto para encontrar compañeros de viaje, intercambiar consejos y disfrutar de la comunidad.

## Características

- **Publicaciones y Recomendaciones**: Comparte tus aventuras con fotos y consejos útiles.
- **Conexiones Sociales**: Encuentra otros viajeros y mochileros que comparten tus intereses.
- **Solicitar y Ofrecer Aventones**: Conecta con personas que necesitan o pueden ofrecer transporte en sus viajes.
- **Interacción Dinámica**: Da me gusta, comenta y sigue a otros usuarios para mantenerte al día con sus experiencias.
- **Interfaz Intuitiva**: Navega fácilmente a través de un diseño limpio y moderno.

## Tecnologías Utilizadas

- **Flutter**: Para el desarrollo de la interfaz de usuario multiplataforma.
- **Supabase**: Para la gestión de base de datos y autenticación de usuarios.
- **Dart**: Lenguaje de programación utilizado para el desarrollo de la aplicación.

## Instalación

### Prerequisitos

Asegúrate de tener instalado:

- [Flutter](https://flutter.dev/docs/get-started/install) (versión 3.7.0 o superior)
- [Dart](https://dart.dev/get-dart)

### Clonar el Repositorio

```bash
git clone https://github.com/No-Country-simulation/equipo-c24-76-ft-mobile
cd mobile
```

### Instalar Dependencias

```bash
flutter pub get
```

### Ejecutar la Aplicación

Conecta tu dispositivo o emulador, y luego ejecuta:

```bash
flutter run
```

## Uso

1. **Registro/Iniciar Sesión**: Crea una cuenta o inicia sesión en la aplicación utilizando tu correo electrónico o redes sociales.
2. **Crear Publicaciones**: Comparte tus experiencias de viaje con fotos y recomendaciones.
3. **Explorar**: Navega por publicaciones de otros usuarios y conecta con ellos.
4. **Solicitar/Ofercer Aventones**: Utiliza la función de aventones para encontrar acompañantes o ayudar a otros a viajar.
